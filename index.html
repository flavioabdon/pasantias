MQTT

(Message Queue Telemetry Transport)

Sitio Oficial https://mosquitto.org

Ventajas

  • MQTT es un protocolo abierto, sencillo, ligero y fácil de implantar.
  • Especialmente orientado al Iot
  • Utiliza el mínimo de ancho de banda
  • Ideal para utiñizar en redes inalámbricas
  • Requiere pocos recursos procesadores y memorias
  • Consume muy poca energía
  • Es muy rápido y posibilita un tiempo de respuesta superior al resto de protocolos web actuales
  • Permite una gran fiabilidad si es necesario

Como funciona...

Ejemplo publicar y subscribirnos a mqtt mediante una terminal

Para subcribirnos

			      
$mosquitto_sub -h 'host' -p 'puerto' -t "topico"

Para publicar

			      
$mosquitto_pub -h 'host' -p 'puerto' -t "topico" -m "mensaje"

Que es NodeJs

  • Node es un intérprete Javascript del lado del servidor que cambia la noción de cómo debería trabajar un servidor.
  • Su meta es permitir a un programador construir aplicaciones altamente escalables y escribir código que maneje decenas de miles de conexiones simultáneas en una sólo una máquina física
